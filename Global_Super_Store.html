This project is my personal project where I have created Interactive Hardware sales report using Microsoft Powerbi. Here I have followed some steps such as:

1. Connecting DB
2. Analyzing Tables & Relations
3. Data Cleaning - Power Query Editor, DAX
4. Developing Model
5. Creating report.


Project Objective:

Track the sales of global superstore, the data set is an open-source dataset provided by power bi.
I would like to thank End to end Youtube Channel for publishing a portfolio project for beginners.

Project Description:

The objective can be broken down into the following detailed component.
1. Dashboard creation: Identify the KPIâ€™s, design an intuitive and visually appearing dashboard, and interactive visualization and filtering capability to allow users to explore the data at various levels of granularity.
2. Data analysis: Provide valuable insights regarding the superstore sales.
3. Actionable Insights: The goal is to provide the real scenario of the business, comparison of sales quarter to quarter. With the help of this dashboard stakeholders can find out the profitable products, and loss-making products, Average delivery time and number of product returns, what precautions can be taken to increase the sales and business development.

Insights Drawn:

1. The total sale from year 2012 till 2015 is 12.64 M, the toral revenue is 986.57M and the total quantity sold is 178K.
2. The average delivery time is 4 days, and the total returned products are 1079.
3. Canon Image class is the highest profitable product.
3. The top 5 Customers data is very similar to each other.
4. Consumer Segment holds 51.48%, Corporate segment is 30.25% and Home Office 18.27% of sales.
5. there are several loss-making products which needs to be sold quickly and be careful regarding purchasing/manufacturing them in the future.
