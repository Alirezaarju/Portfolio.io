This project is my personal project where I have created Interactive Data Professional Survey Breakdown report using Microsoft Powerbi. Here I have used data cleaning methodology, ETL process and data analysis techniques to draw the conclusion from the data.


Project Objective:

To find out the result of Data Professional Survey. I would like to thank Alex, The Analyst for the awesome project idea and providing the data set. and Enterprise DNA YouTube channel for publishing a way to use image as slicer.

Project Description:

The objective can be broken down into the following detailed component.
1. Dashboard creation: Identify the KPI’s, design an intuitive and visually appearing dashboard, and interactive visualization and filtering capability to allow users to explore the data at various levels of granularity.
2. Data analysis: Provide valuable insights regarding Data related career. Average Salary for different roles, Work/Life balance, difficulty to break into data, most used programming language, Satisfaction level with the remuneration etc.
3. Actionable Insights: The goal is to provide the career guidance to the data enthusiasts. Undaughterly Data is the future, there are many roles available for the data enthusiasts such as Data Science. Data Engineering, Data Analysis and many more. One can break rock and grab the dream job if he/she is properly skilled.

Insights Drawn:

1. Average Salary for Male and females are nearly same, male(m – 49.2%), (F – 50.8%).
2. The Average salary for Data scientists is always higher, Data Analyst position comes at 5.
3. 60% of the Data analyst are happy with their Work/Life Balance and 44.9% of Data Analyst are happy with their current salary.
4. Python is used more frequently by all data professionals. Data Analyst use Python the most among all other.
5. 45% of the Voter think, getting into the Data field is neither easy nor difficult.
